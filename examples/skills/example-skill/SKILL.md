# Example Skill

This is an example skill that demonstrates the basic structure of a structured_skills skill.

## Purpose

This skill provides example functionality for testing and demonstration purposes.

## Usage

When using this skill, you can:

1. List available resources
2. Run the example scripts
3. Read the skill content

## Available Functions

- `greet(name: str)` - Returns a greeting message
- `calculate(a: int, b: int)` - Performs a calculation

## Notes

This is a demonstration skill and should not be used in production.
